# TradeSellBuy (TSB)

TSB is a vendor e-commerce website that is built using React + Laravel . 

# Features

-   Authentication
    -   Login with email and password
    -   Register with email and password
    -   Logout
    -   Update user profile
    -   Update password
    -   Delete account
    -   Role base authentication 
-   Roles
    - Customer
        - Default Role 
        - View Products
        - View User Profiles
        - Add Products to cart
    - Seller
        - Seller Role
        - Sell Products
        - Dashboard to edit and view your own products
    - Admin
        - Admin Role
        - Assign and Remove Roles
        - View all Users
        - View all Orders
-   Reviews
    -   Write a Review to a user
    -   Edit Review
    -   Delete Review
    -   Give Rating 
    -   Show total Ratings of a user
-   Favourites
    -   Add a Product to your favourites
    -   Remove a Favourite Product
    -   View Favourites
